Detailed Description

A model is defined by his mesh and his textures. It can load and draw a mesh and textures.

Member Enumeration Documentation

◆ TYPE_FILE

enum Model::TYPE_FILE

Define the type of the mesh to load :

  • OBJ (.obj) : classic mesh,
  • MNT (.asc) : a height map of a mountains
  • NONE : just for load a hard code mesh (DEBUG).
