class  Scale
 This class manage one scale. A scale is composed of an height map and all the texture generated from it. all scale have the same size (width, height) based on the DEM (.asc) in input. Use : create the scene (don't forget use generateHeightMap if you use the create function with no heightmap) generate the texture with a generate function send a generated texture to a shader with sendMapToShader function or draw only one texture with draw function.
