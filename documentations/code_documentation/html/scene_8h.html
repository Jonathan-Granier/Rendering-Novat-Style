class  Scene
 This class manage the stack of Scale ( on set of texture) , the mesh and all the texture. The goal of this class is manage the multi-scale rendering (it's a Laplacian pyramid). So the main object of this class is _scalesManager . It's a vector that contains scale , their ID and if they are enabled. Show Scale description for more details about the scale.
